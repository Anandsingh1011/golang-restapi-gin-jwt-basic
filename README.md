# Golang Gin Framework - Video API 

Golang REST API with basic jwt authentication using gin framework

## Run

```bash
go run server.go
```

## Endpoint  

```Endpoint 
POST   /login
GET    /api/videos
POST   /api/videos 
```

### Login Details 

```Login Details 
Username : vuser
Password : newpassword
```

### Login  

```login 
POST   /login

Request
---


curl --location --request POST 'http://localhost:5000/login' \
--header 'Content-Type: application/x-www-form-urlencoded' \
--data-urlencode 'username=vuser' \
--data-urlencode 'password=newpassword'
```
